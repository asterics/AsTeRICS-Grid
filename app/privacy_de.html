Verarbeitung von Daten

AsTeRICS Grid ist eine Webapplikation für Unterstützte Kommunikation (UK). Die Oberfläche kann individuell an die Bedürfnisse von NutzerInnen angepasst werden. Es können eigene Bilder und Texte in benutzerdefinierten Raster hinterlegt werden. Weiters gibt es die Möglichkeit von mitlernenden Wörterbüchern zur Texteingabe, welche häufig verwendete Wörter und Sätze speichern. Diese Daten werden nachfolgend zusammenfassend "personalisierte Daten" bzw. "personenbezogene Daten" genannt. Es gibt 2 Arten AsTeRICS Grid zu nutzen, wobei die Daten jeweils verschieden gespeichert werden.

Offline-User

Bei einem Offline-User werden alle personalisierten Daten nur auf dem Gerät der NutzerInnen gespeichert und nicht ins Internet übertragen. Die Daten werden in einer internen Datenbank des Browsers (IndexedDB) gespeichert, auf die nur der aktuelle Nutzer des Geräts Zugriff hat.

Online-User

Bei einem Online-User werden alle personalisierten Daten in einer Datenbank der AsTeRICS Foundation gespeichert und können somit auch von anderen Geräten der NutzerInnen abgerufen und synchronisiert werden. Alle personalisierten Daten werden Ende-zu-Ende verschlüsselt. Das heißt, dass nur der Nutzer selbst sie unverschlüsselt sehen kann, wennn er sich mit seinen Zugangsdaten anmeldet. Die AsTeRICS Foundation hat keine Möglichkeit zum Zugriff oder zur Weiterverarbeitung der personenbezogenen Daten, welche in der Nutzerkonfiguration verschlüsselt gespeichert werden. Der Schlüssel zum Verschlüsseln der Daten ist das vom Nutzer selbst gewählte Passwort. Zum Verschlüsseln wird der Algorithmus AES verwendet und die Softwarebibliothek sjcl.

Folgende Daten können in der Datenbank der AsTeRICS Foundation ohne Schlüssel zum Entschlüsseln eingesehen werden:

  • Benutzername: der frei wählbare Benutzername wird unverschlüsselt in der Datenbank der AsTeRICS Foundation gespeichert. Der Benutzername wird nur zur Identifikation des Nutzerkontos verwendet. Der Benutzername ist der einzige Datensatz, welcher unter Umständen einer Person zugeordnet werden kann. Er kann aber vom Nutzer so gewählt werden, dass er völlig anonym ist.
  • Anzahl der Grids: obwohl alle Inhalte verschlüsselt sind, kann in der Datenbank nachvollzogen werden, wie viele Grids (Raster) ein Nutzer angelegt hat.

Zugriffsanalyse

Wir verwenden für AsTeRICS Grid die Besucheranalyse-Software von Matomo. Alle dadurch erfassten Daten sind auf unseren eigenen Servern gespeichert, werden nicht weitergegeben und lassen keine Rückschlüsse auf einzelne Personen zu. Folgende Daten werden gespeichert:
  • Anonymisierte IP-Adressen der Benutzer (die letzten 2 Bytes werden gelöscht, z.B. 81.217.xxx.xxx)
  • Anzahl und Dauer des Zugriffes
  • Betriebssystem, Browser-Version und Bildschirmauflösung, die für den Zugriff verwendet wurde

Verwendung von YouTube

Sobald im eingebauten YouTube Player ein Video gestartet wird, werden Cookies von YouTube angelegt. Wenn Sie den eigebauten YouTube-Player verwenden möchten, müssen Sie den YouTube Nutzungsbedingungen und der Google Datenschutzerklärung zustimmen. Alle eingegebenen Suchanfragen für YouTube-Videos werden an die YouTube Data API gesendet um Suchergebnisse zu erhalten und passende Videos abspielen zu können.

Rechtsgrundlage

Die Rechtsgrundlage für die Verwendung des Benutzernames (einziger von AsTeRICS Grid verarbeiteter unter Umständen personenbezogener Datenwert) ist Einwilligung, da bei der Registrierung für einen Online-User ein entsprechendes Häkchen zu setzen ist.

Datenübertragung

Es werden keine Daten an Dritte weitergegeben.

Speicherdauer

Alle Benutzerdaten von Online-Usern werden nach 365 Tagen ohne Verwendung (kein Login des Nutzers) gelöscht. Soll ein Online-User zuvor gelöscht werden, so ist dies durch eine Anfrage an office@asterics-foundation.org möglich.

Widerruf und Informationen zu Rechten

Die Einwilligung zur Verarbeitung der Daten kann jederzeit durch eine Anfrage an office@asterics-foundation.org widerrufen werden. Im Falle des Widerrufs werden die Daten des entsprechenden Users gelöscht.

Weitere Informationen zu Ihren Rechten (Auskunftsrecht, Löschung, Beschwerde usw.) finden Sie auf der Info-Seite zum Datenschutz der FH Technikum Wien. Bei Fragen können Sie sich jederzeit an office@asterics-foundation.org wenden.

Wenn Sie Bedenken zur Rechtsmäßigkeit der Verarbeitung der Daten haben, können Sie die Österreichische Datenschutzbehörde kontaktieren.
